# Contributing to Global Managed Aquifer Recharge Screener

Thanks for your interest in improving the Global Managed Aquifer Recharge Screener.

## What contributions are most useful?

Priority contributions include:

- Field hydrogeological observations that help interpret screening outputs.
- MAR case studies that can test or refine the intervention-setting logic.
- Regional adaptation notes for climate, terrain, and recharge context.
- Documentation improvements, bug reports, and wrapper-site UX suggestions.

## How to contribute

1. Open an issue describing the problem, idea, or dataset.
2. If you have a code or content change, create a branch and submit a pull request.
3. Keep changes focused and explain the reason for the update clearly.
4. Where possible, include evidence, references, or screenshots.

## Reporting issues

Please include:

- What page or link you were using.
- What you expected to happen.
- What actually happened.
- Browser and device details if the wrapper page is involved.
- Country or area of interest if the issue relates to the GEE app output.

## Important note

This repository wraps a screening-level Earth Engine app. Contributions should preserve the core warning that the outputs are not a substitute for field investigation, aquifer testing, water-quality assessment, or engineering design.
