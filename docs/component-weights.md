# Default Weighting Notes

The Global Managed Aquifer Recharge Screener uses transparent expert-judgement defaults intended for screening and prioritisation, not statistically calibrated prediction.

## Main idea

The weighting configuration favours tributary and footslope settings because these are often actionable in decentralised, semi-arid, rural, and basement-complex contexts.

## Important caution

These defaults should be treated as a starting hypothesis. They should be reinterpreted or recalibrated when local hydrogeological evidence becomes available.

Examples of useful local evidence include:

- Borehole performance and groundwater levels
- Geophysical interpretation
- Regolith thickness and lithology maps
- Flood and runoff behaviour
- Existing MAR scheme performance

## Why keep the weights transparent?

Clear, interpretable defaults help users understand what the model is rewarding and what it is penalising. That makes the tool easier to critique, adapt, and improve responsibly.
